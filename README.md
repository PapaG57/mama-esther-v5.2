# 🌿 Mama Esther — Plateforme associative React + Vite

Bienvenue dans le projet web de l'association **Mama Esther**, une initiative solidaire centrée sur l'accueil, l'aide et le soutien aux personnes vulnérables à Saint-Nicolas en Forêt 🇫🇷

Ce site a été conçu avec ❤️ et React/Vite pour offrir :

- 💌 Un espace de contact chaleureux
- 🧵 Une vitrine pour les actions de terrain
- 🎨 Un design doux et cohérent, aux couleurs de l'humanité

---

## ✨ Technologies utilisées

- ⚛️ **React** avec **Vite** pour la performance et le développement rapide
- 💨 **Tailwind / CSS maison** pour le style
- 📬 **Formspree** pour les formulaires sécurisés
- 🔒 Respect des données avec RGPD intégré
- 🔧 Composants réutilisables façon Mama-style

---

# 🌿 Mama Esther — Nonprofit platform using React + Vite

Welcome to the web project of the **Mama Esther** nonprofit, a community initiative focused on welcoming, helping, and supporting vulnerable people in Saint-Nicolas-en-Forêt 🇫🇷

This site was designed with ❤️ and React/Vite to offer:

- 💌 A welcoming contact space
- 🧵 A showcase for grassroots initiatives
- 🎨 A soft and cohesive design, in the colors of humanity

---

## ✨ Technologies Used

- ⚛️ **React** with **Vite** for performance and rapid development
- 💨 **Tailwind / Homemade CSS** for styling
- 📬 **Formspree** for secure forms
- 🔒 Data compliance with integrated GDPR
- 🔧 Reusable components, Mama-style
